# Scar Media Hub

منصة متكاملة لإدارة وعرض الوسائط الرقمية (أفلام، مسلسلات، ألعاب، انمي، مسرحيات، قرآن) مع متجر تخزين مدمج.

## 🚀 التقنيات

| التقنية | الاستخدام |
|---|---|
| **TanStack Start** | إطار العمل الرئيسي (SSR + Routing) |
| **React 19** | واجهة المستخدم |
| **Tailwind CSS 4** | التنسيق والتصميم |
| **SQLite (better-sqlite3)** | قاعدة البيانات المحلية |
| **Supabase** | قاعدة بيانات احتياطية/سحابية |
| **TanStack Query** | إدارة حالة البيانات من السيرفر |
| **Zustand** | إدارة حالة السلة (Cart) |
| **shadcn/ui** | مكتبة المكونات |
| **Lucide Icons** | الأيقونات |
| **Vite 7** | أداة البناء |
| **TypeScript** | لغة البرمجة |

## 📁 هيكل المشروع

```
src/
├── components/       # مكونات واجهة المستخدم
│   └── ui/          # مكونات shadcn/ui
├── hooks/           # React Hooks مخصصة
├── integrations/    # تكاملات خارجية (Supabase)
│   └── supabase/
├── lib/             # دوال مساعدة ومنطق السيرفر
│   └── server/     # دوال السيرفر (Database, API)
├── routes/          # الصفحات (File-based Routing)
│   ├── admin/      # لوحة الإدارة
│   └── ...
└── styles.css       # التنسيقات العامة
```

## 🛠️ التشغيل

```bash
# تثبيت الاعتماديات
npm install

# تشغيل بيئة التطوير
npm run dev

# بناء للإنتاج
npm run build

# معاينة البناء
npm run preview
```

## 🌟 المميزات

- **تصفح الوسائط**: عرض وتصفية الوسائط حسب التصنيف، التصنيف الفرعي، اللغة، والسنة
- **السلة**: إضافة الوسائط إلى سلة الشراء مع حساب الحجم الإجمالي
- **متجر التخزين**: شراء هاردات وفلاشات مدمجة مع الطلب
- **لوحة الإدارة**: إدارة كاملة للميديا، السلاسل، الطلبات، المستخدمين، والعروض
- **نظام العروض**: خصومات على الحجم أو السعر تلقائياً
- **API**: واجهة برمجية للوصول إلى الميديا والسلاسل
- **نظام الدفع**: دعم طرق دفع متعددة مع محافظ إلكترونية

## 🗄️ قاعدة البيانات

قاعدة البيانات SQLite محلياً في مجلد `data/`:
- `media.db` — بيانات الوسائط والسلاسل
- `orders.db` — بيانات الطلبات
